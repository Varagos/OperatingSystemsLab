### Basic chatting tool

Two-way communication over TCP/IP, using BSD Sockets Api with no message encryption.

```bash
$ git clone <repo-url>
$ cd basic-chat-tool
$ make

# Run server
$ ./server_z1 <PORT>
# Run client
$ ./client_z1 <SERVER_HOST> <SERVER_PORT>

# Exchange messages between them
```
